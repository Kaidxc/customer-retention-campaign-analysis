# Customer Retention and Campaign Targeting Analysis

## Project overview

This is an independent SQL and Power BI portfolio project.

The project analyses customer purchasing behaviour for a UK online retailer
and identifies high-value customers who should be prioritised for a retention campaign.

## Business problem

The marketing team has a limited campaign budget and cannot contact every inactive customer.

The goal of this project is to answer:

- Which customers are most valuable?
- Which customers appear at risk of becoming inactive?
- Which customers should be prioritised for a retention campaign?
- How should the campaign impact be measured?
